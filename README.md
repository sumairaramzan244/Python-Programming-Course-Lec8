## Lecture 8 Task — Strings in Python
In this practice task, I learned how strings work in Python. Strings are immutable which means once created they cannot be changed. I practiced different string operations like concatenation, indexing and slicing.

---

## What I Learned  
* Strings are immutable — once created they cannot be changed  
* `+` operator is used to join two or more strings together — this is called concatenation  
* `.upper()` converts all letters of a string to uppercase  
* `.lower()` converts all letters of a string to lowercase  
* `len()` function returns the total number of characters in a string  
* Indexing is used to access a single character  `name[0]` gives the first character  
* Negative indexing starts from the end — `name[-1]` gives the last character   
* Slicing is used to get a part of a string — `name[3:6]` gives characters from index 3 to 5    
* `name[:6]` slices from the beginning up to index 5
